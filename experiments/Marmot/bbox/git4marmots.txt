Load the changes by others

	git pull origin master
	
Add a file

	git add <filename>
	
Remove a file

	git rm <filename> 
	
  or	 

	git rm -f <filename> 
	
Broadcast the changes

	git commit -m "<message>"	
	git push origin master
	
  and then visit  github.com/tallmarmot/pgf and press "New pull request"
